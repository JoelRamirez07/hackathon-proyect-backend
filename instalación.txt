1. Instalación de Python
Para iniciar el sistema, es necesario instalar Python. En esta ocasión, se utiliza la versión 3.11.9. Python puede instalarse en cualquier ubicación seleccionada por el usuario.

2. Creación del Entorno Virtual
Una vez instalado Python, se debe crear un entorno virtual. Para ello, abra CMD y acceda a la carpeta donde se trabajará. Luego, ejecute los siguientes comandos:
python -m venv venv
venv\Scripts\activate

Si la activación es correcta, se visualizará la carpeta del entorno virtual precedida por (venv) en la terminal:
(venv) C:\ruta\a\proyecto>

3. Instalación de Dependencias
En la raíz del proyecto, asegúrese de contar con un archivo requirements.txt que contenga las siguientes librerías:
fastapi
uvicorn
requests
beautifulsoup4
openai
llama-index
chromadb
python-dotenv

4. Configuración de la API
Originalmente se utilizaba la plataforma OpenAI, pero se ha reemplazado por Gemini. Los pasos son los siguientes:
Acceda a: AI Studio – Gemini
Obtenga su API Key en la sección Get API Key.
Coloque la clave obtenida en el archivo .env del proyecto.
Instalar la librería correspondiente en el entorno virtual:
pip install google-genai

Con el entorno virtual activado, instalar las dependencias ejecutando:
pip install -r requirements.txt

5. Ejecución del Proceso de Ingesta
Para ejecutar el proceso inicial, utilice:
python partida_a/test_ingesta.py

Adicionalmente, instalar las siguientes librerías necesarias para procesamiento de modelos:
pip install torch transformers sentence-transformers

Luego, instalar ChromaDB:
pip install chromadb

6. Configuración de FastAPI
Para levantar la API utilizando FastAPI, siga estos pasos:
Crear el archivo main.py en la carpeta correspondiente.
Instalar FastAPI y Uvicorn en el entorno virtual:
pip install fastapi uvicorn

Ejecutar el servidor de desarrollo:
uvicorn main:app --reload
